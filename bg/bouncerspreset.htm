Game Courier Preset for Bouncers


The following preset enables you to play Bouncers per e-mail, using a graphical interface. To move a bouncer, type, for example, B c2-d3 (use b for black). When making multiple bounce moves, you don't need to specify the intermediate bounce moves. When promoting a bouncer, type, for example, B b9-a10; add K a10 (use small letters for black). When capturing an enemy bouncer, type, for example, B b7-c8; capture b d9 (small letters denote black pieces). This preset does not enforce rules.

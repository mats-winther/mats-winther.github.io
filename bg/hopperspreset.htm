Game Courier Preset for Hopper Dame


The following preset enables you to play Hopper Dame per e-mail, using a graphical interface. To move a hopper, type, for example, H c2-e4 (use h for black). When making multiple jumps, you don't need to specify the intermediate jumps. When promoting a hopper, type, for example, H b9-a10; add K a10 (use small letters for black). This preset does not enforce rules, so you have to understand the rather unusual promotion rules.
